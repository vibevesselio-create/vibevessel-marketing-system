This repository contains the VibeVessel Marketing System codebase, including:

- **Google Apps Scripts (GAS):** DriveSheetsSync, Project Manager Bot, and other automation scripts

- **Agent Coordination System:** Multi-agent coordination and handoff management
- **Documentation:** System documentation, agent protocols, and operational guides

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/vibevesselio-create/vibevessel-marketing-system.git
   cd vibevessel-marketing-system
   ```

2. Set up environment:
   - Copy `.env.example` to `.env` (if available)
   - Configure required environment variables
   - Install dependencies as needed

## Important Notes

- **Secrets:** Never commit secrets, API tokens, or credentials
- **Environment Files:** All `.env*` files are excluded via `.gitignore`
- **State Files:** `var/state/**/*.json` files are excluded to prevent secret leakage
